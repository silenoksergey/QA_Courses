# В рамкар курса Вадима Ксендзова был проведеён ступенчатый тест в Jmeter с помощью плагина Stapping Thread Group
## Тест показал следующие границы производительности сервера:
1)Стабильная работа без ошибок при 79 пользователей
2)Критические ошибки сервера - количество ошибок до 3% при 98-99 пользователей
3)Краш сервера - количество ошибок больше 3% при 100 пользователей с удержанием нагрузки более 10 секунд
